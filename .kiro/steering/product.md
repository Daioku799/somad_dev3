# Product Overview

3D-ICの積層構造における熱解析モデルの構築と最適化を支援するツール。複雑な半導体設計データを、幾何学的に安定した解析用モデルへと変換する。

## Core Capabilities

- **外部データ連携**: JSONおよびGDSII形式からモデルパラメータ（寸法、物性、レイアウト）を柔軟に読み込む。
- **動的モデリング**: 層厚や成分配置の変更に対し、Z座標マーカーやメッシュ境界を自動再計算する。
- **安定マッピング**: 幾何プリミティブ（直方体、円柱、球）を用いた厳密な図形判定により、格子生成の失敗を排除する。
- **検証視覚化**: 解析用モデルの正当性を確認するための多断面プロットおよび解析結果の等高線描画。

## Target Use Cases

- **TSV配置最適化**: TSVの本数やピッチを設計変数として、熱特性を最適化するための反復的なモデル生成。
- **スタック設計評価**: 層数や材料構成を変更した際の熱影響を迅速に評価するプロトタイピング。

## Value Proposition

実設計の複雑さをプリミティブな幾何形状に抽象化することで、格子生成の安定性と計算速度を確保し、設計変数を外部ファイルで容易に変更できる「実験環境」を提供する。
